A interface está implementada no terminal, logo qualquer ação será realizada
com comandos, esfecificados na tela ou no menu.

Nesta aplicação são permitidas ações de empréstimo e devolução de livros,
sendo que essas operações estão sujeitas à penalidades, caso os usuários 
cadastrados não respeitem a data de devolução do livro.

Pode-se cadastrar usuários, cujos campos são nome, limite de tempo com os 
livros, tipo de usuário pode este ser aluno, professor ou comum, e o número
de livros de cada tipo de usuário pode emprestar por vez.

Para o cadastro de livros, temos os campos de título, e auxiliares para o 
tipo , texto ou não, e se ele está alugado.

Com esse sistema pode-se registrar empréstimos, devoluções, mostrar registros
com atrasos na entrega, e cálculo da penalidade por não devolver os livros.
Além disso, para melhor gerenciamento, há como listar todos os registros
